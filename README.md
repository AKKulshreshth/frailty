This is an early version of a frailty screening dynamics model. You can only run the model with Stella Architect. Without this software you can view the pdfs and get a sense of the logical structure. You can also view all equations in the txt file.

  
You need to have the Stella model file (Frailty 101 v0Q.stmx), the input file (Frailty 101 inputs.csv) and output files (Frailty 101 Outputs 1.xlsx and Frailty 101 OUtputs 2.xlsx) to run the model.

  
The Frailty 101 Inputs.xlsx file is for ease of changing input values. Do *not* link the model input to this file. The tab 1. Frailty_inputs_do_not_edit consolidates all the needed inputs. Save this tab as "Frailty 101 Inputs.csv" if you do make changes.

  
Use Model->Import Data ("Source") to link to the input file.

  
Model->Export Data to create links to the two output files.
