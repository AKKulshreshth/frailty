"incidence_rate_pre-frail_to_frail"[gender, age] = IF (age_as_a_number[age] >= 50) THEN  (1 - (EXP("OR_pre-frail_to_frail_Alpha"[gender] -"OR_pre-frail_to_frail_Beta"[gender]*age_as_a_number[age]) / (1 + EXP("OR_pre-frail_to_frail_Alpha"[gender] - "OR_pre-frail_to_frail_Beta"[gender]*age_as_a_number[age])) )^(1/5)) ELSE 0
    UNITS: Dimensionless
    DOCUMENT: See documentation for incidence rate robust to pre-frail
"incidence_rate_robust_to_pre-frail"[*, *] = IF (age_as_a_number[age] >= 50) THEN  (1 - (EXP("OR_robust_to_pre-frail_Alpha"[gender] - "OR_robust_to_pre-frail_Beta"[gender]*age_as_a_number[age]) / (1 + EXP("OR_robust_to_pre-frail_Alpha"[gender] - "OR_robust_to_pre-frail_Beta"[gender]*age_as_a_number[age])) )^(1/5)) ELSE 0
    UNITS: Dimensionless
    DOCUMENT: Data sources
    =============
    
    Wei, Kai, et al. "Nutritional and frailty state transitions in the Singapore Longitudinal Aging Study." The Journal of nutrition, health and aging 22.10 (2018): 1221-1227.
    
    Cheong, C. Y., et al. "Risk factors of progression to frailty: findings from the Singapore longitudinal ageing study." The Journal of nutrition, health and aging 24.1 (2020): 98-106.
    
    
    
    Base transition rates:
      Robust → Prefrail : 43.4% over 5 years  (Wei et al. 2018, Table 3, SLAS-2)
      Prefrail → Frail  : 22.28% over 4.5 years (Cheong et al., Results, SLAS-2)
    
    Age and gender log-odds coefficients (borrowed from Cheong et al., same SLAS-2 cohort):
      β_age    = ln(1.07) = 0.06766 per year of age
      β_female = ln(0.91) = −0.09431 (female indicator)
    
    LOGISTIC FRAMEWORK
    -------------------------
    For each transition, the cumulative probability P over the follow-up horizon T is:
    
      logit(P) = ln[P/(1−P)] = α + 0.06766 × age − 0.09431 × female
    
    where female = 1 for women, 0 for men, and α is a transition-specific intercept.
    
    INTERCEPT DERIVATION — Robust → Prefrail
    --------------------------------------------------
    Anchored at Wei Table 1 Robust subgroup means:
      Mean age     = 64.8 years
      Prop. female = 355/570 = 62.3%
      Base rate    = 43.4% over 5 years
    
      logit(0.434) = ln(0.434/0.566) = −0.266
    
      Covariate contribution C = 0.06766 × 64.8 − 0.09431 × 0.623
                               = 4.384 − 0.059 = 4.325
    
      α = −0.266 − 4.325 = −4.591
    
    Gender-specific intercepts (absorbing β_female × female):
      Male   (female=0): α_male   = −4.591
      Female (female=1): α_female = −4.591 − 0.09431 = −4.685
    
    Verification: at age 64.8, 62.3% female →
      logit(P) = −4.591 + 4.384 − 0.059 = −0.266 → P = 43.4% ✓
    
    INTERCEPT DERIVATION — Prefrail → Frail
    ---------------------------------------------
    Anchored at Cheong et al. sample means:
      Mean age     = 65.62 years
      Prop. female = 63.9%
      Base rate    = 22.28% over 4.5 years
    
      logit(0.2228) = ln(0.2228/0.7772) = −1.249
    
      Covariate contribution C = 0.06766 × 65.62 − 0.09431 × 0.639
                               = 4.439 − 0.060 = 4.379
    
      α = −1.249 − 4.379 = −5.628
    
    Gender-specific intercepts:
      Male   (female=0): α_male   = −5.628
      Female (female=1): α_female = −5.628 − 0.09431 = −5.722
    
    Verification: at age 65.62, 63.9% female →
      logit(P) = −5.628 + 4.439 − 0.060 = −1.249 → P = 22.28% ✓
    
    ANNUAL TRANSITION RATE FORMULA
    -------------------------------
    The logit equation gives cumulative probability P over horizon T (5 or 4.5 years).
    Assuming constant hazard, the annual rate r is:
    
      r(age, gender) = 1 − (1 − P(age, gender))^(1/T)

    ----------------------------------------
    OR_robust_to_prefrail_Alpha[male]   =  4.591
    OR_robust_to_prefrail_Alpha[female] =  4.685
    OR_prefrail_to_frail_Alpha[male]    =  5.628
    OR_prefrail_to_frail_Alpha[female]  =  5.722
    OR_robust_to_prefrail_Beta[male]    =  0.0677
    OR_robust_to_prefrail_Beta[female]  =  0.0677
    OR_prefrail_to_frail_Beta[male]     =  0.0677
    OR_prefrail_to_frail_Beta[female]   =  0.0677
    
    Note: Alpha values are stored as positive numbers in the model.
    The Stella formula uses them as (Alpha − Beta × age), matching
    the sign convention logit(P) = α − β × age, where the negative
    sign on Beta reflects increasing transition probability with age.
    
    IMPORTANT: the model uses these estimates above age 50, and this age threshold is hard-coded!

    EXTENSION TO MODEL AGE BANDS
    -----------------------------
    The model covers single-year age cohorts from NewBorn to Age100andOver,
    with frailty stocks active from age 55 onwards (Age_55_plus_vector = 1
    for ages 55-100+, 0 otherwise).
    
    Age 0-54  : initial_frailty_prevalence = 0
                Initial_prefrailty_prevalence = 0
    
    Age 55-59 : No Vaingankar data available (survey started at age 60).
                Assumption: prevalence equals the 60-74 band values,
                i.e., frailty = 3.5%, pre-frailty = 37.9%.
                Rationale: frailty is rare below 60 and the 60-74 band
                is already dominated by the 60-64 sub-group, making it
                a conservative upper bound for ages 55-59.
    
    Age 60-74 : initial_frailty_prevalence      = 0.035
                Initial_prefrailty_prevalence   = 0.379
    
    Age 75-84 : initial_frailty_prevalence      = 0.123
                Initial_prefrailty_prevalence   = 0.475
    
    Age 85+   : initial_frailty_prevalence      = 0.246
                Initial_prefrailty_prevalence   = 0.534
    
    NORMALISATION
    -------------
    The three prevalence values (robust + pre-frail + frail) must sum
    exactly to 1.0 for each cell. Checking against Vaingankar:
    
      60-74 :  0.586 + 0.379 + 0.035 = 1.000 ✓
      75-84 :  0.402 + 0.475 + 0.123 = 1.000 ✓
      85+   :  0.220 + 0.534 + 0.246 = 1.000 ✓
    
    No normalisation adjustment required.
    
    APPLICATION IN STELLA
    ----------------------
    These prevalence values are applied uniformly across all race and
    gender dimensions within each age band, as Vaingankar reports no
    statistically significant race differences in frailty prevalence
    (p = 0.022 at bivariate level but not retained in the adjusted
    regression), and no significant sex difference in frailty
    (p = 0.082, Table 1).
    
      INIT frail_undiagnosed    = resident_population * initial_frailty_prevalence
      INIT pre-frail_undiagnosed = resident_population * Initial_prefrailty_prevalence
      INIT robust_people        = resident_population * (1 - initial_frailty_prevalence
                                                           - Initial_prefrailty_prevalence)
    
    The diagnosed stocks (frail_diagnosed, pre-frail_diagnosed) are
    initialised at zero, consistent with the assumption that no
    screening programme existed at the 2010 simulation start date.
